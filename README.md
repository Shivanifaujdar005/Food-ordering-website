# 🍔 Food Ordering Website

## Overview

A responsive food ordering website developed using HTML, CSS, and JavaScript. The project provides an engaging user interface for browsing food items, exploring menu categories, and viewing customer reviews. The website is designed with a mobile-friendly layout to ensure a seamless experience across different devices.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Attractive and modern user interface
* Food menu with multiple categories
* Customer reviews section
* Smooth navigation and interactive elements
* Optimized layout for improved user experience

## Technologies Used

* HTML5
* CSS3
* JavaScript
